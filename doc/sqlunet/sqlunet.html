Features

  • Support for MySql, SQlite.

WordNet features

  • Complete port (however, orphaned morphological forms are dropped)
  • Retains synset index as primary key allowing easy reference to wordnet original database
  • Includes WordNet 2.0 to 2.1, 2.1 to 3.0, 2.0 to 3.0 sense maps
